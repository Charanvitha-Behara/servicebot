An AI-powered educational service bot that helps students with academic queries like **exam dates** and **syllabus**.

## Features
- Flask backend
- MongoDB storage
- NLP-based query understanding
- Web chat UI

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Seed MongoDB: `mongo < database/seed_data.js`
3. Run app: `python app.py`
4. Open browser at `http://127.0.0.1:5000/`
